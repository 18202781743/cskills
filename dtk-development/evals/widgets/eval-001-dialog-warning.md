# Eval: 创建带警告样式的对话框

## 任务

使用 DTK 创建一个确认删除对话框，要求：
1. 标题为"确认删除"
2. 消息为"此操作不可撤销，确定继续吗？"
3. 包含两个按钮：红色"删除"按钮和普通"取消"按钮
4. 返回用户点击的按钮索引

## 期望输出

代码应：
1. 使用 DDialog 类
2. 使用 DDialog::ButtonWarning 类型创建警告按钮
3. 使用 exec() 获取用户选择
4. 正确处理按钮返回值

## 验证要点

- [ ] 使用 #include <DDialog>
- [ ] addButton 参数包含 ButtonWarning
- [ ] 检查 exec() 返回值判断用户选择
