# Eval: 实现可勾选菜单项

## 任务

用户需要在右键菜单中添加可勾选的选项。

指导用户配置 isCheckable 和 checked 字段。

## 期望输出

回答应：
1. 展示 isCheckable: true 的菜单项
2. 展示 checked 字段设置当前状态
3. 展示 invokedMenuItem 中处理 checked 参数
4. 说明更新菜单状态的方式

## 验证要点

- [ ] isCheckable: true
- [ ] checked: 当前状态
- [ ] invokedMenuItem 中 checked 参数反映勾选状态
- [ ] 状态变化后重新生成菜单数据
