# 主题系统

DTK 主题系统包含调色板（DPalette）、图标（DDciIcon）、字体和风格（Chameleon），实现 Light/Dark 双主题自动切换。

## 触发关键词

主题切换、调色板、DPalette、DGuiApplicationHelper、亮色/暗色、dci 图标、ColorSelector
