# Plugin Structure

## Directory Layout

Every dde-shell plugin follows this structure:

```
<plugin-name>/
├── CMakeLists.txt                    # Build configuration
├── <name>applet.h                    # C++ class header
├── <name>applet.cpp                  # C++ class implementation
├── package/
│   ├── metadata.json                 # Plugin metadata (required)
│   ├── main.qml                      # QML entry point
│   └── control/                      # Custom QML controls (optional)
├── configs/
│   └── <plugin-id>.json              # DConfig metadata (optional)
└── translations/                     # Translation files
    ├── <plugin-id>.ts                # Base translation
    ├── <plugin-id>_zh_CN.ts          # Chinese (Simplified)
    ├── <plugin-id>_zh_HK.ts          # Chinese (Hong Kong)
    ├── <plugin-id>_zh_TW.ts          # Chinese (Taiwan)
    └── ... (23 languages total)
```

## Required Files

### metadata.json

The plugin metadata file that defines the plugin identity:

```json
{
    "Plugin": {
        "Version": "1.0",
        "Id": "org.deepin.ds.example",
        "Url": "main.qml",
        "Parent": "org.deepin.ds.dock"
    }
}
```

Fields:
- `Version`: Plugin version (string)
- `Id`: Unique plugin identifier (reverse domain notation)
- `Url`: QML entry file relative to package directory
- `Parent`: Parent plugin ID (for containment hierarchy)
- `ContainmentType`: Type indicator for panels (e.g., "Panel")

### CMakeLists.txt

Build configuration that:
1. Creates the shared library
2. Links against `dde-shell-frame`
3. Installs the package using `ds_install_package()`

### C++ Class

The plugin's C++ class that:
- Inherits from `DApplet`, `DContainment`, or `DPanel`
- Implements `init()` for initialization
- Uses `D_APPLET_CLASS()` macro for registration

## Optional Files

### DConfig Configuration

Place in `configs/` directory with the plugin ID as filename:
```json
{
    "magic": "dsg.config.meta",
    "version": "1.0",
    "contents": {
        "settingName": {
            "value": "defaultValue",
            "serial": 0,
            "flags": [],
            "name": "Setting description",
            "permissions": "readwrite",
            "visibility": "public"
        }
    }
}
```

### Translation Files

Generated by `ds_handle_package_translation()` macro. Supports 23 languages including:
- ar (Arabic), az (Azerbaijani), bo (Tibetan), ca (Catalan)
- de (German), es (Spanish), fi (Finnish), fr (French)
- hu (Hungarian), it (Italian), ja (Japanese), ko (Korean)
- lo (Lao), nb_NO (Norwegian), pl (Polish), pt_BR (Portuguese)
- ru (Russian), sq (Albanian), uk (Ukrainian)
- zh_CN, zh_HK, zh_TW (Chinese variants)

## Installation Paths

After build and install:
- Package: `/usr/share/dde-shell/<plugin-id>/`
- Plugin library: `/usr/lib/dde-shell/plugins/`
- Translations: `/usr/share/dde-shell/<plugin-id>/translations/`
